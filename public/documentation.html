My Movie API Documentation

Business Logic URL HTTP Method Request body data format Response body data format
All movies /movies GET none A JSON object holding data about all the movies
Single movie by title /movies/[title] GET none A JSON object holding data about a single movie, containing description, genre, director, image URL.
Example: 
{
    _id: ObjectId('66ffd38a0cd7d2bec4c73c0a'),
    title: 'Chasing Dreams',
    description: "A heartwarming story about a young artist's journey to find her voice and inspire others.",
    imagePath: '/images/chasing_dreams.jpg',
    featured: 'false',
    genre: {
        name: 'Fantasy',
        description: 'A genre that features magical elements, mythical creatures, and fantastical worlds.'
    },
    director: {
        name: 'Christopher Nolan',
        bio: 'TEST An acclaimed director known for his complex narratives and innovative filmmaking techniques. His works often explore themes of time, memory, and reality.',
        birthDate: ISODate('1970-07-30T00:00:00.000Z'),
        dearthDate: null
    },
    bio: 'TEST An acclaimed director known for his complex narratives and innovative filmmaking techniques. His works often explore themes of time, memory, and reality.'
}
Single genre by name /movies/genre/[name] GET none A JSON object holding data about a single movie genre, containing name and description
Example: 
{
    "name": "Fantasy",
    "Description": "Fantasy Movie"
}
Single director by name /movies/director/[name] GET none A JSON object holding data about a single movie director data
Example: 
{
    "name": "Zack Snyder",
    "Bio": "Bio fo Director",
    "Birth": "xx"
}
New user register /users POST A JSON object holding data about the user to add, structured like: 
{
    "userName": "newUser1",
    "email": "username@mail.com",
    "password": "password",
    "firstName" : "firstName",
    "lastName": "lastName",
    "birthDate": "1986-04-16T00:00:00.000Z"
}
A JSON object holding data about the user that was added, including an ID: 
{
    "userName": "newUser1",
    "password": "password",
    "firstName": "firstName",
    "lastName": "lastName",
    "birthDate": "1986-04-16T00:00:00.000Z",
    "createdDate": "2024-10-05T13:58:22.912Z",
    "favorite": [],
    "_id": "670145fe918c4b4a7c137739",
    "__v": 0
}
Update user data /users/[username] PUT 
{
  "userName": "newUser2",
  "email": "username@mail.com",
  "password": "password",
  "firstName" : "firstName",
  "lastName": "lastName",
  "birthDate": "1986-04-16T00:00:00.000Z"
}
A text message indicating the user data was updated
Add movie to user's favorites list /users/[username]/favorite/[movieid] PUT none A text message indicating the user favorites list is added with the movie
Delete movie from user's favorites list /users/[username]/favorite/[movieid] DELETE none A text message indicating that the movie is removed from the user favorites list
User deregister /users/[username] DELETE none A text message indicating that the user has been deregistered
