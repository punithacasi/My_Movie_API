Single genre by name and movie title /movies/[title]/genre/[name] GET none A JSON object holding data about a single movie genre, containing name and description
Example: 
{
    "name": "Fantasy",
    "Description": "Fantasy Movie"
}
Director by movie title /movies/[title]/director GET none A JSON object holding data about a single movie director data
Example: 
{
    "name": "Zack Snyder",
    "Bio": "Bio fo Director",
    "Birth": "xx"
}
New user register /users POST A JSON object holding data about the user to add, structured like: 
{
    email : "testname@mail.com" 
    name : "Test User"    
}
A JSON object holding data about the user that was added, including an ID: 
{
    id:1,
    email : "testname@mail.com" 
    name : "Test User"
}
Update user data /users/[email]/[username] PUT none A text message indicating the user data was updated
Add movie to user's favorites list /users/[email]/[movietitle] PUT none A text message indicating the user favorites list is added with the movie
Delete movie from user's favorites list /users/[email]/[movietitle] DELETE none A text message indicating that the movie is removed from the user favorites list
User deregister /users/[email] DELETE none A text message indicating that the user has been deregistered
